\doxysubsection{Detailed Description}
Definition of the return values and the function prototypes. 

Base code for Unit Testing.

In order to avoid unwated exceptions and errors during all reading operations, the functions in this file are defined in order to return the correct values and the associated error codes if they were to fail. All of this having in Unit Testing.
